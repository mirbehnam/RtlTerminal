English

Rtl Terminal for Windows

Rtl Terminal is a free and open-source Windows terminal emulator by behnamapps. It is designed for Persian, Arabic and other right-to-left language users while remaining useful for standard English command-line work.

The application uses Windows ConPTY and supports Unicode text, terminal colors, progress bars, scrollback history, clickable links, keyboard shortcuts and customizable fonts.

فارسی

ترمینال فارسی و راست‌به‌چپ برای ویندوز

Rtl Terminal یک ترمینال رایگان و متن‌باز برای ویندوز است که توسط behnamapps ساخته شده است. این برنامه امکان نمایش کامل راست‌به‌چپ را برای متن‌های فارسی و عربی فراهم می‌کند.

برنامه از CMD، پاورشل، WSL، Bash، رنگ‌های ANSI، نوار پیشرفت، لینک‌های قابل کلیک، سابقه اسکرول و انتخاب فونت‌های نصب‌شده ویندوز پشتیبانی می‌کند.
